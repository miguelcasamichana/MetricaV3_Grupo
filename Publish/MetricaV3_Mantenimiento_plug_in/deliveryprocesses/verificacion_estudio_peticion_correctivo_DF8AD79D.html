Main Description

Esta tarea indica como se debe verificar la información a la hora de juzgar si se ha de atender la petición o no.

Ya que hay dos formas de hacer mantenimiento, esta tarea se indica de forma separada al caso evolutivo.

Steps
Verificar si la información es correcta
Reproducir el problema
Comienzo de estudio: se establece hasta que punto es un problema crítico
Si es crítico se comienza analisis y solución inmediatamente
Si no es crítico se clasifica para resolverlo cuando se tenga la solución más adecuada
